Anthropic「Claude が自社プロダクションコードの80%以上を執筆」― 再帰的自己改善の現実と AI 一時停止ボタン構想

June 4, 2026 開発者向け エージェント

Anthropic が 6 月 4 日に公開した「When AI builds itself」によると、 2026年5月に Anthropic の本番コードベースへマージされたコードの 80% 以上が Claude によって執筆された。 2025年2月のリサーチプレビュー開始時点では AI 執筆率は一桁台パーセントだった。 同期間にエンジニア一人あたりのコードマージ量は 8倍に増加し、 複雑な未定義タスクにおける Claude の成功率は6か月で 26% → 76% に上昇。 Anthropic はこの「再帰的自己改善」の加速を受け、各国が検証可能な形で フロンティア AI 開発を一時停止できる グローバル・ポーズボタン 構想を提唱した。

エンジニア的注目点: 自社のコード生成 AI 活用率・マージ量の変化を計測していないなら今すぐ始めるべき。エンジニア生産性指標が「コミット数→AI マージ量」に移行しつつある現実に合わせ、KPI の再設計を検討すること。

GitHub Copilot がトークン課金制へ完全移行 ― 月額 $29 が $750 に跳ね上がるケースも、開発者コミュニティが猛反発

June 1, 2026 開発者向け

GitHub Copilot が 6 月 1 日から プレミアムリクエスト単位（PRU） を廃止し、 消費トークン数に応じた AI Credits 課金 へ切り替えた。 ベースプラン価格（Pro $10、Pro+ $39、Business $19、Enterprise $39）は据え置きだが、 ヘビーユーザーでは月額が $29 → $750、 場合によっては $50 → $3,000 に急増したという報告が Reddit・X で拡散。 コード補完と Next Edit Suggestions は引き続き無制限だが、 高性能モデルを多用するチャット・エージェント機能が課金対象となる。 移行先として Claude Code・Windsurf・オープンソース代替 の評価が急増している。

エンジニア的注目点: Copilot のヘビーユーザーは今すぐトークン消費の実測値を確認すること。GPT-4o ベースの機能を多用しているチームは Claude Code や Codeium への乗り換えコスト計算が急務。

Google Gemini 3.5 Flash 正式公開 ― フラッグシップ級の精度を 4 倍速・$1.50/M トークンで提供、Gemini 2.0 系は全廃

June 1, 2026 新モデル 開発者向け

Google が Gemini 3.5 Flash を正式リリース。 従来の大型フラッグシップモデルと同等の性能を 4 倍の速度 で実行し、 価格は 入力 $1.50 / 出力 $9 per million トークン。 6 月 1 日をもって Gemini 2.0 系は全廃となり、3.x が新しいベースラインとなった。 Google インフラを使うすべてのアプリケーションは自動的に 3.x ベースに移行するため、 既存の Gemini 2.0 ベースの実装は API レスポンスの変化を確認する必要がある。 長文コンテキスト処理・マルチモーダル対応も強化されており、 RAG パイプラインや文書解析ユースケースでの優位性が際立つ。

エンジニア的注目点: Gemini 2.0 系の廃止は静かな破壊的変更。Google AI Studio または Vertex AI の管理コンソールで今すぐ使用モデルバージョンを確認し、3.5 Flash への移行テストを走らせること。

Claude Code が「最も愛される AI コーディングツール」46% 獲得 ― 開発者認知が1年で 31% → 57% に急伸

June 2026 開発者向け エージェント

Pragmatic Engineer が 906 人のソフトウェアエンジニアを対象に実施した調査（2026年2月）によると、 Claude Code が AI コーディングツールの「最も愛されている」部門で 46% を獲得し首位。 開発者への認知率は 2025年中頃の 31% から 2026年1月に 57% に急上昇し、 職場での採用率は同期間で約 6 倍に成長した。 ターミナルネイティブの操作性・ファイルシステムへの直接アクセス・Claude 3.x 系モデルとの シームレスな統合が高評価の要因として挙げられている。 一方で Copilot の課金変更を受け、Copilot から Claude Code への移行を評価するチームが急増している。

エンジニア的注目点: Claude Code の認知・採用急拡大はツール乗り換えコストの低さと裏表。チームに Claude Code 未経験者がいるなら、今週 /init コマンドで CLAUDE.md を生成する体験を全員にしてもらうのが最速のオンボーディング。

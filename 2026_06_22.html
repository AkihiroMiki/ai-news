xAI Grok Imagine Video 1.5 GA ― I2V アリーナ首位、Sora 2 Pro の 86% 安

Jun 16, 2026 クリエイティブ 新モデル

xAI が Grok Imagine Video 1.5 を 6 月 16 日に正式 GA（一般提供）へ移行。 Image-to-Video Arena の Elo ランキングで首位に立ち、 API 価格は 720p で $4.20/分（Sora 2 Pro の $30/分に対して 86% 安）。 主な改善点はサウンド（音響・音声を映像と同一パスで生成）、モーション（クリップ全体での物理的一貫性向上）、速度（6秒・720p を約 25 秒で生成、前世代比 37% 高速）の 3 点。 Grok.com・iOS・Android アプリと API の全チャネルで利用可能になった。 また、プロジェクト単位で複数エージェントを並列起動する新ワークフロー機能も同時リリース。

エンジニア的注目点: Sora 2 Pro と同等クラスの品質で 1/7 のコストは動画生成機能のプロダクト組み込みを現実的な選択肢に変える。$4.20/分という単価で自社サービスの動画生成 API コストを試算し直す価値がある。

Perplexity が「Brain」を発表 ― AI エージェントが一晩かけて自分の記憶を更新する仕組み

Jun 18, 2026 エージェント

Perplexity が Computer エージェント向けの自己改善型メモリシステム Brain を発表。 セッション・コネクター結果・ドキュメント変更・ユーザーの修正指示を元に 夜間バッチで LLM Wiki を自動更新する設計で、 エージェントが「何がうまくいったか・何が失敗したか」を次のセッションに持ち越せる。 文脈依存タスクで回答正確率 +25%、再現率 +16%、 コスト -13% の改善を確認。 ユーザーはメモリの内容を閲覧・削除でき、各メモリのソース（セッション・ファイル・コネクター）が明示される。 Perplexity Max サブスクライバー（$200/月）向けのリサーチプレビューで提供開始。

OpenAI LifeSciBench 公開 ― PhD 173 人が設計した 750 問の生命科学ベンチマーク、AI は 3 問に 1 問しか通過しない

Jun 17, 2026 新モデル 開発者向け

OpenAI が LifeSciBench を 6 月 17 日に公開。 バイオテクノロジー・製薬分野の PhD 173 人が設計・レビューした 750 タスクで構成され、 実際の研究者が日常的に行う「データ付き実験設計・根拠の扱い・科学的コミュニケーション」を問う。 自社モデル GPT-Rosalind はテキストのみのタスクで 45.1% を通過する一方、 データファイルが添付されるタスクでは 28.1% まで 17pt 低下する。 AI は「きれいな教科書問題」には強いが、実験データを含む現場の仕事では依然として弱いことが示された。

AWS FinOps Agent プレビュー ― コスト質問に答え・最適化提案・Jira チケット起票まで自律処理

Jun 15〜16, 2026 エージェント 開発者向け

AWS が FinOps Agent のプレビューを公開。 「先月どのサービスが一番コストかかった？」「この EC2 最適化できる？」といった質問に自然言語で応答し、 コスト異常の調査、最適化提案の提示、さらに Jira チケットの自動起票まで一連の処理を行うエージェント。 従来は Cost Explorer でデータを引き出し、分析してチケットを起票するフローを人が担っていたが、 その一連を AI エージェントに委任できる設計。 AWS Bedrock 上で動作し、既存の AWS ロール・権限体系と統合される。

エンジニア的注目点: クラウド費用の管理は「誰がやるかわからず後回し」になりがちな業務。FinOps Agent のようなドメイン特化エージェントが Bedrock 上で動くことで、AWS ロール管理のノウハウがあれば追加実装なしに運用できる。大規模 AWS 利用組織は早期プレビュー参加を検討する価値がある。

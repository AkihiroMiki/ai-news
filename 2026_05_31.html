Anthropic「Code w/ Claude London」― Managed Agents セルフホストサンドボックス公開ベータ & MCP トンネル研究プレビュー

May 26, 2026 エージェント 開発者向け

Anthropic が 5 月 26 日に開催した Code w/ Claude London イベントで、 Claude Managed Agents の重要アップデートを発表。 エージェントのツール実行を企業自身のインフラ上で動かせる セルフホストサンドボックス（公開ベータ） が解禁され、 データ主権・コンプライアンス要件を持つ企業でも本番導入が現実的になった。 さらに MCP サーバーをセキュアなトンネル経由で接続する MCP Tunnels（研究プレビュー） も公開。 プライベートネットワーク内の社内ツールにエージェントからアクセスできる経路が整備された。

エンジニア的注目点: セルフホストサンドボックス + MCP Tunnels の組み合わせで「クラウドに出さずに動く完全社内エージェント」が構築可能に。既存の VPC / オンプレ環境との統合設計をすぐ始めるべきタイミング。

Anthropic が Add-ins・Dreaming・Outcomes を正式発表 ― エージェント設計の新プリミティブ
